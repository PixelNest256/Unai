DDGS Search Skill Help

Usage:
- "search [keyword]" - Search keyword and display summary
- "find [keyword]" - Search keyword and display summary  
- "look up [keyword]" - Search keyword and display summary
- "ddgs [keyword]" - Search keyword and display summary

Examples:
- "search python programming"
- "find latest AI news"
- "look up weather tokyo"
- "ddgs machine learning"

Features:
- Uses DuckDuckGo search engine to search for information
- Returns summary of search results
- Displays title and body of first search result
